# Football Registration Form

Football Registration Form is a registration form which helps in registering a new user.
##### Here is the UI look:
<img width="658" alt="image" src="https://user-images.githubusercontent.com/67963905/154211124-98dc4664-73c5-4798-ab78-9c0ad100be54.png">

## What does this form do?
#### The Football Registration Form helps in registering a new energetic player, retrieving his/her information submitted, and updating his/her information accordingly.

## How to access and run this form?
#### To access this form, you need to clone this repository on your system. After being cloned, you need to build this project in IDE like IntelliJ, Eclipse, etc which will install all the dependencies which are required to run this application. The dependencies which are required are Apache Tomcat,SQL Server-JDBC , and Gson. Moreover, MS SQL Server Studio is required to access and view the enteries in the database.
#### After all the setup, you need to click on this link: **[Local Host](http://localhost:8080/index.html)**
